# Python Data Engineering Practice

## Overview
This is a Python learning repository for Data Engineering fundamentals. It contains daily practice scripts covering Python basics, data processing, and related topics.

## Project Structure
- `main.py` - Entry point that runs all practice scripts
- `daily_practice/` - Daily practice Python files covering topics like:
  - Variables and functions
  - Print statements
  - Operators
  - Conditionals (if-else)
  - Lists
  - Logging
- `assignments/` - Assignment files for practice exercises

## How to Run
Use the "Run Practice Scripts" workflow to execute all practice scripts, or run individual files with:
```bash
python daily_practice/<script_name>.py
```

## Language & Environment
- Python 3.11

## Topics Covered
- Python basics (variables, lists, conditions, loops)
- Functions & modules
- File handling (CSV, JSON)
- Logging & error handling
- Date & time processing
- Python + SQL (SQLite)
- Pandas basics
